# Personal Site

A minimal personal website built with React and Tailwind CSS.

## Features

- Clean, modern design
- Dark mode with automatic system preference detection
- Social media links (Spotify, Apple Music, YouTube, Letterboxd)
- Responsive layout
- Smooth animations

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
